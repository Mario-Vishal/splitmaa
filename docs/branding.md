# Branding

Splitmaa means Split + Gemma.

The current branding foundation is intentionally minimal:

- Warm off-white background.
- Peach/orange accent.
- Geometric split-circle logo built with `react-native-svg`.
- Wordmark component in the mobile app.
- Light and dark color token structure.

The logo is a code component, not a final app-store icon. The generated Expo template assets are still placeholders until Phase 1 polish replaces them.

Splitmaa is an independent open-source project and is not affiliated with Google or the Gemma team.
